# Sales Forecasting Using Machine Learning

## 📌 Project Overview
This project focuses on building a **Sales Forecasting System** using Machine Learning techniques.

Sales forecasting helps businesses predict future sales trends based on historical sales data. Accurate forecasting enables better decision-making in inventory management, marketing strategies, and business planning.

This project uses historical **Superstore sales data** to analyze sales trends and forecast future sales.

---

## 🎯 Objectives
The project aims to:

✔ Analyze historical sales data  
✔ Identify monthly sales trends  
✔ Visualize sales performance over time  
✔ Build a Machine Learning forecasting model  
✔ Predict future sales values

---

## ⚙️ Project Workflow

### 1. Data Loading
- Imported Superstore sales dataset
- Loaded CSV file into Pandas DataFrame

### 2. Data Preprocessing
Performed preprocessing tasks:

- Converted `Order Date` to datetime format
- Checked missing values
- Cleaned and prepared data

### 3. Monthly Sales Analysis
Grouped sales data by month to analyze trends over time.

### 4. Data Visualization
Created visualizations including:

- Monthly Sales Trend Graph
- Historical Sales Analysis
- Forecast Visualization

### 5. Model Training
Built a **Linear Regression model** for sales forecasting.

Steps included:

- Train-test split
- Model training
- Sales prediction

### 6. Model Evaluation
Evaluated model performance using:

- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)

### 7. Future Sales Forecasting
Predicted future sales trends and visualized forecasted sales.
